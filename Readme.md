# Tunguska
Tunguska is a UCI-compliant chess engine written in C++11.

## Compiling
>g++ -std=c++11 -O2 -DNDEBUG *.cpp -o tunguska_1_1

## Engine Strenght
It plays around 2400-2450 ELO.
